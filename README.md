# Webserv

Um projeto educacional da **42SP** que implementa um servidor HTTP completo do zero, explorando conceitos fundamentais de redes, arquitetura de software, programação orientada a eventos e design de sistemas escaláveis.

## Milestones
- [Fundação](docs/milestones/fundacao.md)
- [Servidor Estático](docs/milestones/servidor_estatico.md)
- [Servidor Dinâmico](docs/milestones/servidor_dinamico.md)
- [Validação e Demonstração](docs/milestones/validacao_e_demonstracao.md)


> **Status**: 🚧 Em desenvolvimento | **Linguagem**: C++ (C++98) | **Plataforma**: Linux

## 🎯 Objetivo do Projeto

Construir uma base sólida e extensível para um servidor HTTP robusto, aplicando:

* ✅ **Arquitetura em camadas** com separação clara de responsabilidades
* ✅ **Design Patterns** estabelecidos (Singleton, Observer, Factory, Builder)
* ✅ **Programação orientada a eventos** com multiplexing I/O (epoll/poll/select)
* ✅ **Boas práticas de engenharia** (SOLID, Clean Code, testes)
* ✅ **Conformidade HTTP/1.1** e suporte a recursos modernos (IPv6)
* ✅ **Configuração estilo Nginx** com parser robusto
